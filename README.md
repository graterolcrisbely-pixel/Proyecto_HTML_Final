# EcoMarket

EcoMarket es una tienda virtual enfocada en la venta y promoción de productos ecológicos, reutilizables y sostenibles. Su objetivo es fomentar el consumo responsable y el cuidado del medio ambiente mediante una experiencia web sencilla e intuitiva.

## Descripción

Este proyecto fue desarrollado utilizando tecnologías web básicas:

- HTML5
- CSS3

La página permite a los usuarios conocer la empresa, explorar un catálogo de productos ecológicos y comunicarse mediante un formulario de contacto.

---

## Características

- Página principal con presentación de la empresa.
- Sección de productos ecológicos.
- Página "Sobre Nosotros" con historia, misión y visión.
- Formulario de contacto.
- Diseño responsive básico.
- Animaciones y estilos personalizados.
- Navegación entre páginas.

## Páginas del Sitio

### Inicio
Página principal donde se presenta EcoMarket, promociones destacadas y acceso rápido al catálogo.

### Productos
Catálogo con productos ecológicos y reutilizables disponibles para los usuarios.

### Sobre Nosotros
Información acerca de la empresa, su historia, misión y visión.

### Contacto
Formulario para que los usuarios puedan enviar mensajes o solicitudes.

---

## Objetivo del Proyecto

Promover el uso de productos sostenibles y crear conciencia sobre la importancia de cuidar el medio ambiente mediante alternativas ecológicas para el día a día.
